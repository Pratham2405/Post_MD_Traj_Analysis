# Post_MD_Traj_Analysis
This is a repository for the implementation of Linear MD trajectory methods TICA(time-aware), PCA and RMSF.
Convert the cartesian trajectory file(.xtc and .gro) into dihedral features by using `dihedrals_from_gro_xtc.py`.
If you want to run the three utilities with one script, use `run_linear_analyses.sh`.
